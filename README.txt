the goal of this app is to incorporate various components that will help in time management
the app will be, primarily, a To Do list for the person to track what needs to be done for that day
in addition, the app will be an Eisenhower Matrix so the user can prioritize what work needs to be done
the app will pull in meetings from Outlook Calendar into the ToDo list so the user can see what meetings need to be done
the app will have a simple timer to help the user keep on task with their work

the meetings from outlook will be pulled into the app
-only Accepted meetings will be pulled in (not Tentative meetings)
-The meetings will be put into Q1 of the To Do/Eisenhower matrix (Important and Urgent)
